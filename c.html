O que é C++

C++ é uma linguagem de programação compilada multi-paradigma e de uso geral. Desde os anos 1990 é uma das linguagens comerciais mais populares, sendo bastante usada também na academia por seu grande desempenho e base de utilizadores.

Prós Contras
O programa C++ tem muitas vantagens quando envolve programação. Todos os arquivos autônomos do programa C++ devem usar a função principal para permitir que o programa inicie e motive suas funções. Um grande problema em C++ é quando a informação aponta para algo equivalente a partir de dois pontos de partida diferentes, isso causa um problema sério, o programa C++ ainda terá problemas envolvidos na codificação.
Os dados globais e as funções globais são usados ​​em C++ que não são usados ​​em muitas outras linguagens de alto nível dentro das ciências do computador e é uma vantagem para as linguagens de programação. O código único de Java para as regras ASCII é de 16 bits, enquanto o programa C++ é apenas de 8 bits. Portanto, C++ pode ser uma linguagem de programação menos impressionante, mas economiza memória.
O programa C++ é útil para linguagem de programação de baixo nível e realmente eficiente para fins gerais, oferece desempenho e memória de forma eficiente, oferece abstração de alto nível, dentro do domínio da linguagem do assunto O programa C++ é complexo durante um programa de alto nível muito grande, C++ é comumente empregado para aplicativos específicos da plataforma.
