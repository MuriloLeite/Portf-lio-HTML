O que é HTML

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML. HyTime é um padrão para a representação estruturada de hipermídia e conteúdo baseado em tempo. Um documento é visto como um conjunto de eventos concorrentes dependentes de tempo, conectados por hiperligações. O padrão é independente de outros padrões de processamento de texto em geral.

Prós Contras
Mais rápido para editar: para edições simples, geralmente é mais rápido fazer alterações em uma página usando um editor de texto. Deve-se saber HTML: embora a maioria dos editores de texto HTML possa ajudar com tags e sugerir atributos e assim por diante, esses auxiliares não substituem o conhecimento de HTML. A maioria dos editores de texto modernos oferece estilos de arrastar e soltar, como negrito e itálico, mas se você não se lembrar do código para “espaço ininterrupto”, seu editor pode não ser capaz de ajudar.
Mais comercializável: Um desenvolvedor da Web que pode escrever HTML usando um editor de texto será mais comercializável do que aquele que só pode usar um editor WYSIWYG. O primeiro é mais flexível e pode ficar atualizado em qualquer ferramenta de edição de HTML, enquanto o último tem que começar tudo de novo a cada nova ferramenta de edição. Curva de aprendizado mais íngreme: Como você precisa aprender tanto HTML quanto as funções do editor em si, um iniciante achará um editor de texto mais difícil de usar.
HTML legível por humanos: isso é especialmente importante se você trabalhar em uma equipe de desenvolvedores da web. O HTML pode ser espaçado como sua equipe gosta e incluir comentários ou outras notas para permitir uma edição mais eficiente por outros membros da equipe. Mais difícil de “projetar” com: Algumas pessoas acham os editores de texto mais difíceis de criar páginas porque não conseguem visualizar a aparência da página apenas com o HTML.
